BrokenLinkVerifier 0.9

search brocken links on a html page.

  --help  **prints this usage text**

  -p \<value\> | --port <value>  **smtp port**  
  -t \<to\>,\<to\>... | --to \<to\>,\<to\>...  **mail to**  
  -f \<value\> | --from \<value\>  **mail from**
